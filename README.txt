Projeto referente ao SkyGarden, uma plataforma de hortas horizontais para um público metropolitano.

O projeto foi desenvolvido utilizando Kotlin para o app, e .NET para o desenvolvimento da API. 

Temos muitas melhorias em mente para nosso projeto, como cadastro das plantas, monitoramento inteligente, perfis de usuários diferentes, dentre muitas outras .

Esperamos que gostem! :D
